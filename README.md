# CULTIVAR

VR gardening game prototype developed for an MSc dissertation.

## Download (Meta Quest APK)
Download the latest standalone build from **Releases**:
- Open **Releases** (right sidebar)
- Download `cultivar.apk`

Tested on:
- Meta Quest 2
- Meta Quest 3

## Repository contents
- `Scripts/` – Core Unity C# scripts (gameplay + synchronization logic)
- `Docs/` – System diagrams (architecture, plot lifecycle, sequence diagrams)

## Notes
- This repository intentionally includes **scripts and documentation assets only** (no full Unity project, scenes, or third-party assets).
- The APK is large because it includes the full standalone build content.
